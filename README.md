# Template para Deploy de Produção em Servidores VPS

Este repositório é o material de apoio oficial para o tutorial onde
transformamos um VPS "zerado" em
[uma infraestrutura de produção completa](https://youtu.be/yxxEk68EDgo), segura
e automatizada. Basicamente, seu serviço de `cloud` particular.

Aqui você vai aprender a criar um ambiente com **HTTPS automático (SSL)**,
**Deploy Contínuo (CI/CD)** via GitHub Actions, e segurança de nível
empresarial, tudo rodando sobre um
[KVM 2 - Hostinger](https://hostinger.com/otaviomiranda).

Também detalhei este processo em vídeo caso queira assistir:

- [Crie Sua Cloud em VPS](https://youtu.be/yxxEk68EDgo)

---

## Onde hospedar?

Se você busca um servidor **robusto, confiável e com preço imbatível**, minha
recomendação é o [KVM 2 da Hostinger](https://hostinger.com/otaviomiranda).

**Bônus Exclusivo:** Consegui **10% de desconto adicional** para você. Basta
usar o cupom abaixo no carrinho:

- Cupom: `OTAVIOMIRANDA`

Segundo a própria **Hostinger**, isso é por tempo limitado.
