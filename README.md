# Guia Definitivo: Deploy Profissional com Docker no VPS Hostinger

Este repositório é o material de apoio oficial para o tutorial onde
transformamos um VPS "zerado" em uma infraestrutura de produção completa, segura
e automatizada.

Aqui você vai aprender a criar um ambiente com **HTTPS automático (SSL)**,
**Deploy Contínuo (CI/CD)** via GitHub Actions, e segurança de nível empresarial
— tudo rodando sobre a performance incrível do **KVM 2 da Hostinger**.

---

## Onde hospedar?

Se você busca um servidor **robusto, confiável e com preço imbatível**, minha
recomendação número 1 é o
[KVM 2 da Hostinger](https://hostinger.com/otaviomiranda).

Testei exaustivamente e a performance do KVM 2 (AMD EPYC + NVMe + banda larga)
superou expectativas, sendo perfeito para rodar nossa stack Docker.
